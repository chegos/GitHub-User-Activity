# 🚀 GitHub Activity CLI

Uma aplicação simples em Python que permite visualizar as atividades recentes de qualquer usuário do GitHub diretamente pelo terminal.

## 📌 Funcionalidades

- Buscar atividades recentes de um usuário do GitHub
- Exibir eventos como:
  - Push de commits
  - Criação de branches
  - Abertura de issues
  - Stars em repositórios
- Interface simples via linha de comando (CLI)
- Tratamento de erros (usuário inválido ou falha na API)
